## Synopsis

The old iCopter game in its ASM version.

## Compilation & Running

Compiled using Turbo Assembler (for x64 machines, I highly recommend [DosBox][ln1]).

```sh
c:\> tasm.exe copter.asm
c:\> tlink.exe copter.obj
c:\> copter.exe
```
